# Proyecto: Retail Analytics Pipeline
## (Big Data)

El objetivo de este proyecto es diseñar e implementar un proyecto integral que comience con la ingesta y procesamiento de datos masivos, continúe con la
transformación y análisis mediante Apache Spark, y finalice con la construcción de un modelo de Machine Learning escalable que permita
clasificar y generar información accionable para el área de marketing de una empresa de e-commerce.

El proyecto se desarrolla en 5 etapas evolutivas: 

1. Fundamentos de Big Data.
2. Apache Spark - Introducción y Configuración.
3. Elementos básicos de Spark (RDD, Transformaciones y Acciones).
4. Procesamiento de datos estructurados (Spark SQL y DataFrames).
5. Introducción a Machine Learning Escalable (Spark MLlib).

## Tecnologías utilizadas
* Python (PySpark)
* Apache Spark
* Spark MLlib
* Seaborn
* Matplotlib
* Google Colab
